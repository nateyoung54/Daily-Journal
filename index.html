Delivery setup

Telegram + GitHub Actions
  1. Message @BotFather in Telegram and create a bot.
  2. Send your bot one message, then open the Telegram getUpdates URL to find your chat.id.
  3. Add GitHub secrets named TELEGRAM_BOT_TOKEN and TELEGRAM_CHAT_ID.
  4. Add GitHub variables named DAILY_PROMPT_TIME and DAILY_PROMPT_TIMEZONE.
  5. Run the Veloura Daily Prompt workflow manually once to test delivery.
